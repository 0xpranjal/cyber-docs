## How to get `App ID`

- Generate RSA private key

```bash
openssl genrsa -out privateKey.pem 2048
```

- Generate RSA public key

```bash
openssl rsa -in privateKey.pem -pubout -out publicKey.pem
```

- Copy the public key content. Example:
