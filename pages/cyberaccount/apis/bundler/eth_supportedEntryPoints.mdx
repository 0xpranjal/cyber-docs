# eth_supportedEntryPoints

Returns the list of supported entry points.

request example

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "eth_supportedEntryPoints",
  "params": []
}
```

response example

```json
{
  "id": 1,
  "jsonrpc": "2.0",
  "result": ["0x5FF137D4b0FDCD49DcA30c7CF57E578a026d2789"]
}
```
