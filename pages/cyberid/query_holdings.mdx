# Query CyberID Holdings


## GraphQL

request example
```graphql
query {
  wallet(address: "0xD9c208dB8d08365916C35473b97A390dDAA551B2", chainId: 10) {
    cyberIds {
      totalCount
      pageInfo {
        hasNextPage
        hasPreviousPage
        startCursor
        endCursor
      }
      edges {
        node {
          tokenId
          name
          metadata {
            key
            value
          }
        }
        cursor
      }
    }
  }
}
```
